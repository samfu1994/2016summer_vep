This is a tool to help visualize the features generated by DocuScope.
You should first click the button below "select file" on the left, and select a file you want to see.
You can sort the plots by the criteria you select, also you can switch between narrow mode and normal mode by clicking the red button.
You can also choose the plot you want by clicking the green button.
